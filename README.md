# 📌 Overview

Skin Disease Detection System is a Deep Learning based web application that classifies different skin diseases from uploaded images.

The application uses **EfficientNetV2B0** architecture trained on a curated skin disease dataset and deployed through a **Flask Web Application**.

This system allows users to upload an image of a skin condition and instantly receive:

- Disease Prediction
- Confidence Score
- Fast Inference Results
- User-Friendly Interface

# ⚙️ Installation

## Clone Repository

```bash
git clone https://github.com/Sprasad-2002/Skin-Disease-Detection-System.git
```

## Navigate to Project

```bash
cd Skin-Disease-Detection-System
```

## Install Dependencies

```bash
pip install -r requirements.txt
```

## Run Application

```bash
python app.py
```

Open Browser:

```text
http://127.0.0.1:5000
```
